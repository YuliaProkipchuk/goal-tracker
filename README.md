# Goal Tracker App
This project is a full-stack application built using the MERN (MongoDB, Express, React, Node.js) stack. It enables users to manage their goals, track progress, and maintain a to-do list where tasks can be grouped by date and even linked to specific goals.
## Features
- **User Authentication**: Register, login, and manage user sessions.

- **Goal Management**: Create, update, and delete goals. Goals include details such as name, description, type, due date, and a plan (a list of steps).

- **Daily Tasks**: Manage tasks grouped by date. Tasks can be marked as complete, and some tasks can be linked to a specific goal.

- **Notes**: Create and update your notes. Organize and search notes for quick access to important information.
